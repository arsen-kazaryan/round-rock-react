# Round Rock Web

Учебный лендинг, созданный на **React + Vite**.

## Технологии

- React
- Vite
- JavaScript
- CSS

## Статус проекта

Сайт сверстан только для **десктопной версии**.  
Адаптивная версия (tablet / mobile) пока не реализована.
